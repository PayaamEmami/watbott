The difference

WatBott is powered by Watson Assistant, which allows the bot to understand a user's intent and respond with a dialogue. This eliminates the need for a user to know any specific commands before interacting with WatBott. That being said, custom commands are often desired for streamers, which is a functionality that is included with WatBott.

Open Source

WatBott is an open source project hosted on GitHub. This project was created with the idea of "This is how chatbots should be." With that vision in mind, an open sourceproject is vital to allow others to help improve WatBott, or inspire other chatbot creators to create something even better; either way, it's a win-win scenario.
